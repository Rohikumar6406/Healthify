Healthify is a scalable, microservices-based healthcare platform designed to deliver intelligent health insights and personalized recommendations. It integrates AI to automate calorie prediction, activity analysis, and health recommendations, significantly reducing manual analysis time while ensuring secure and efficient communication across services.

## Architecture & Services
- User Service
- Activity Service
- AI Service
- API Gateway
- Config Server
- Eureka Server (Service Discovery)

## Features
- AI-driven health insights and recommendations
- Calorie prediction and activity analysis
- Secure authentication and authorization
- Scalable microservices architecture
- Event-driven communication
- Centralized configuration and service discovery
